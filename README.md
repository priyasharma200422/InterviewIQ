# InterviewIQ

An intelligent interview evaluation system that analyzes user responses and provides structured feedback on performance.

## 🔥 Features
- Dynamic interview flow (10 questions)
- Real-time answer evaluation
- Score, communication, confidence analysis
- Strength & weakness detection
- Early exit with partial report
- Clean responsive UI

## 🛠 Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Styling: CSS

## 🎯 How it Works
1. User starts interview
2. Answers questions
3. System evaluates responses using rule-based logic
4. Generates performance report

## 🚀 Future Improvements
- AI integration (OpenAI)
- Voice-based interviews
- Performance analytics dashboard
